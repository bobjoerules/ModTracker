# ModTracker

ModTracker is a web-based tool for monitoring the compatibility of Minecraft mods and datapacks with specific versions of the game. It interfaces with the Modrinth API to provide real-time updates on version support and development progress.

## Features

### Version Tracking
Monitor multiple mods and datapacks simultaneously to see if they have been updated for a selected Minecraft version. The application provides a summary of which items are compatible and which are still pending updates.

### Search and Management
Find content directly through the integrated Modrinth search. Tracked items can be sorted by various criteria including date added, alphabetical order, last update time, and compatibility status.

### Data Portability
Tracked lists can be exported as JSON files for backup or sharing. The application also supports importing these JSON files to restore tracking data.

### Update Timeline
A visual timeline displays when support for specific Minecraft versions was added for tracked items. It supports toggling between initial version support and the latest updates.

### Interface Customization
The application includes multiple visual themes and a background gallery. Users can choose from a selection of provided wallpapers, paintings, and blocks, or upload custom images.
